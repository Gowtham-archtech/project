# Contributing to Project

Thank you for considering contributing to our project! Here are a few guidelines to get you started:

## Getting Started
1. **Fork the repository**: Click the 'Fork' button at the top right corner of the repo page.
2. **Clone your fork**: Run `git clone https://github.com/YOUR_USERNAME/project.git` to clone your fork locally.
3. **Create a branch**: Use `git checkout -b your-feature-branch` to create a new branch for your changes.

## Making Changes
- Be sure to write clear and descriptive commit messages that explain your changes.
- Make sure to follow the project's code style and guidelines.
- Write tests for any new features or changes.

## Submitting Your Changes
1. **Push your changes**: After making your changes and committing them, run `git push origin your-feature-branch` to push your changes to your fork.
2. **Create a pull request**: Go to the original repository and click on 'New Pull Request'. Select your branch and submit.

## Code of Conduct
Please note that this project follows a Code of Conduct. By participating, you are expected to uphold this code.

Thank you for your contribution!
